# 卡牌囚徒 · 完整大纲 v13

---

## 一、基础规则

### 1.1 游戏框架

- 100人被强制拉入"深渊牌局"，共10局
- 任意一局判定失败 → 死亡（从画面中剪切消失，无血无尸体）
- 全部10局通关 → 获得一项能力带回现实
- 主持者：面具（实为庄闲，编号000，上轮胜者选择不回去）
- **核心原则**：系统只宣布最低限度的信息。基础规则（阵营平衡、回合机制、积分体系、道具竞拍规则）一律由玩家在生死之间自行摸索

### 1.2 阵营选择

- 每局两个阵营，5分钟倒计时自由选择
- 阵营平衡铁律：双方各有固定50%上限（100人→各50人）。一方满员后弹回后来者，不能选满的一方
- 0:00未选择 → 系统强制分配
- 胜利条件在阵营选择结束后公布

### 1.3 回合制

- 每局3-4回合（依局型不同）。每回合开始时弹出3张可选能力卡（攻击型/防御型/辅助型），3选1
- 前回合选的能力可在后续回合升级进化
- 每回合之间固定休息5分钟
- 每局游戏开始前固定休息15分钟（用于恢复、道具竞拍、情报交换）

### 1.4 死亡豁免

- 触发方式：玩家死亡时（包括被击杀和失败结算），系统自动扣除 **当局局数 × 500 积分** 作为代价
- 扣除成功 → 豁免死亡，转移至胜方阵营（若为阵营失败结算）或原地存活（若为局内被击杀）
- 扣除失败（积分不足）→ 不可豁免，正常死亡
- **豁免成功后不获得胜方阵营的300积分奖励**——你买回了命，不是赢
- 触发时胸口浮出银光，全场可见
- 代价递增：

| 局数 | 豁免代价 |
|------|---------|
| 第1局 | 500 积分 |
| 第2局 | 1000 积分 |
| 第3局 | 1500 积分 |
| ... | ... |
| 第10局 | 5000 积分 |

- 越往后越贵。前期多攒积分的玩家在后期有更多"免死次数"；前期花光积分的玩家后期无险可避
- 庄闲的逍遥之面自带"规则豁免"可免扣分一次——这是他相对其他玩家的隐藏优势

### 1.5 道具系统

- 每局结束后拍卖3件道具，规则详见 [二、道具清单](#二道具清单)
- 道具存在隐藏"牌路"羁绊，集齐同牌路3件触发完整效果。玩家自行发现，系统不公布

## 二、道具清单

### 2.1 规则

- 每局游戏结束后产生3个拍卖道具，共10局
- 第10局结束后游戏崩碎，实际拍卖9局，共27件拍卖道具
- 每3件道具构成一条完整牌路，集齐触发完整羁绊
- 共10条牌路（9条拍卖集齐 + 庄闲自带1条）
- 拍卖上架顺序随机，每次展示一件。可见类型和效果，不可预知后续
- 道具可被转让、继承。死亡玩家的道具留在原地，可被其他玩家拾取

## 三、积分系统

积分是游戏内唯一通货，跨局累积，可转让给其他玩家。初始值为0。

### 3.1 获取途径

| 途径 | 积分 | 说明 |
|------|------|------|
| **阵营获胜** | +300 | 获胜阵营所有存活成员每人获得300积分 |
| **完成目标任务** | +100~500 | 每局有若干目标任务，完成即获积分。具体分值见各局关卡规则 |

> 注：取消原"伤害输出/击杀贡献/掩护队友"的散装积分模式。积分来源只有两条路——赢，或做任务。

### 3.2 消耗途径

| 途径 | 说明 |
|------|------|
| **道具竞拍** | 最主要消耗。每局竞拍使用积分出价，价高者得 |
| **死亡豁免** | 死亡时自动扣除 当局局数 × 500 积分。扣够 → 存活；不够 → 死亡 |
| **积分拍卖**（第6局） | 直接使用积分购买阵营胜利 |
| **转让** | 可自愿转让给其他玩家（如第4局陈迹转积分给老韩） |

### 3.3 排名机制

- 每局实时更新积分排名，所有玩家可见
- 高排名 → 道具竞拍优先购买力 + 死亡豁免更安全
- 低排名 → 阵营选择时被挤去人少方；豁免代价可能付不起
- 陈迹第1局初始积分为0。所在阵营（破晓者）失败无奖励，靠任务积分（约750）扣豁免后剩余约250。后续通过获胜和任务逐步累积

### 3.4 恶意击杀惩罚

- 击杀同阵营或非同阵营玩家均触发"恶意击杀惩罚"
- 惩罚：扣除击杀者一定比例积分（约200-400积分/次）
- 设计意图：防止"杀穿对面就能赢"的纯暴力解法。击杀净收益可能为负——赢了300积分但扣了400，越杀越穷

### 3.5 积分与死亡豁免的关系

- 死亡豁免不再是系统判定，而是**自动扣分购买**
- 扣分公式：当局局数 × 500
- 第1局豁免需500积分（获胜300 + 做任务才够）。第10局需5000积分（几乎不可能靠积分豁免）
- 这意味：前期靠任务攒分 → 中期有豁免安全垫 → 后期豁免贵到几乎买不起 → 终局靠的是"不被打到"，不是靠买命
- 庄闲的逍遥之面自带"规则豁免"可免扣分一次——这是他比普通玩家多一条命的隐藏优势

## 四、人物设定

*十人各对应一条牌路。*

### 4.1 老韩（韩志远）—— 洞察之途

| 项目 | 内容 |
|------|------|
| 身份 | 45岁，高中数学教师。戴眼镜，镜腿歪了用胶带缠着 |
| 牌路 | 洞察之途：数据记录 → 阵营情报·残片 → 数据核心 |
| 核心特质 | 把恐惧转化为数据。别人尖叫时他开始计算。永远在记录——地上、手臂上、纸上 |
| 欲望 | 解开数据结构。深层：回家告诉女儿小满"爸爸没有逃课" |
| 行为模式 | 话只说给特定人听。拍道具极度克制，三局不出手，只为等一件值得的。说话眼不看人——看数字。但看女儿照片时不是这眼神 |
| 语言特征 | "我算了一个东西。""有一个变量我没法推导。""我继续编数据结构。你们继续活。"——永远不说"我觉得"，说"数据表明" |
| 人物功能 | 陈迹策略的"数据底座"。洞察之途核心持有者。故事的情感代价——他的死告诉读者：规则真实，好人会死，没有隐藏生路 |
| 结局 | 第6局死亡。死前将共有数据用生命灌进卡牌 → "庄家之证"传给陈迹。最后一句："若见小满，告诉她爸爸没有逃课。" |

**写作注意**：永远不说"我觉得"，说"我算过"。眼镜胶带、袖口数字——比"他执着"有力百倍。语气像上课讲题，带商量的尾音但不含糊。死前不写"觉得活不过这局"——让他平常说话，死时读者才被抽巴掌。

### 4.2 白羽 —— 守护之壁

| 项目 | 内容 |
|------|------|
| 身份 | 25岁，急诊科护士。深色长发扎马尾，手很稳 |
| 牌路 | 守护之壁：定身符 → 否决 → 庇护之光 |
| 核心特质 | 嘴硬心软。说"同伴没有意义"，然后用行动窗口掩护陌生人。不高声喊话——低头对最近的人说两句，那人就动了，下一个。急诊分诊的肌肉记忆 |
| 欲望 | 在被迫自私的系统里，不做"推别人去死的类型" |
| 行为模式 | 站的位置永远在能用身体挡住的距禈。手很稳，翻开眼皮还是握道具牌。不说话时比说话时更有存在感 |
| 语言特征 | 短句。"信。""我救人。他们就不让我死。"问题直接："算的还是蒙的？"不说谢谢和对不起——用行动替代 |
| 牌路契合 | 守护之壁三个道具——定身（强制暂停威胁）、否决（取消致命一击）、庇护（持续减伤）——本质是"急诊室逻辑"的卡牌化。她在现实中"争取时间让病人活着"，在游戏里也一样 |
| 结局 | 存活。和陈迹一起回到现实 |

**写作注意**：不让白羽解释感情。"她看着陈迹"足够。嘴硬不戳破——说冷话做暖事，让对比自己站立。医疗背景不出职业说明，通过翻眼皮、搭手腕自然流露。

### 4.3 陈迹 —— 悖逆之印

| 项目 | 内容 |
|------|------|
| 身份 | 21岁，大三学生。成绩中不溜。单机策略游戏玩家。不联机不语音 |
| 牌路 | 悖逆之印：空白牌 → 复制之镜 → 阵营契约 |
| 核心特质 | 不主动，不拒绝。高压下保持观察——不是冷静，是从情感上没投入。策略思维来自一个人面对复杂系统的习惯 |
| 欲望 | 不是"想活下去"，是"不想被规则摆布"。后期：找出面具后的人——不是复仇，是"想知道为什么" |
| 行为模式 | 双手插口袋是默认姿势。说话前有极短停顿——在脑内跑完一轮。能不动就不动，一旦动了就是关键一步。从不解释自己的选择 |
| 语言特征 | 极短。"嗯。""一半一半。"能让别人说的他不说。第5局对全大厅那段是他全篇最长——不是在说话，是在做博弈论题 |
| 牌路契合 | 悖逆之印的道具一开始都是"空"的——空白牌什么都不是，复制之镜本身没有原版能力，阵营契约改变的是别人的身份。共同点：不创造力量，撬动已有的力量。和陈迹一致——不是最强的人，是最会用别人力量的人 |
| 结局 | 存活。口袋里多了两张牌回到现实 |

**写作注意**：不写"感到愤怒/悲伤/恐惧"，用动作。不用"陈迹是那种……的人"。策略推理直接给结果，不配旁白。和白羽对视：看一眼移开，不翻译眼神。

### 5.2 第1局：守护兽之战

**基本信息**
- 阵营：守夜人 vs 破晓者（各50人）
- 胜利条件：击杀对方阵营的守护兽
- 战场：暗红色荒地，双方各有一根巨柱，守护兽从柱底裂缝爬出
- 守护兽：约四层楼高，形态不断变化（骨板→鳞片→黑雾），每回合攻击模式不同
- **本局目标任务**：
  1. 对对方守护兽造成关键伤害 → +200~400
  2. 侦查守护兽攻击模式/弱点并告知队友 → +200
  3. 掩护队友免受守护兽攻击 → +150
- **获胜奖励**：胜方阵营每人 +300

**回合明细**

| 回合 | 可选能力示例 | 陈迹(破晓者) | 白羽(守夜人) | 老韩(破晓者) | 伤亡 |
|------|-------------|-------------|-------------|-------------|------|
| R1 | 破甲一击(攻)/龟壳防御(防)/侦查眼(辅) | 选侦查眼→看己方守护兽→发现积分面板 | 选防御型，组织附近玩家 | 只观察记录，手指在地上写数字 | 破9+守5=14 |
| R2 | 急速(攻)/硬化(防)/弱点标记(辅) | 选辅助型(加队友攻速)。发现后颈破绽窗口→告知建筑工人指挥 | 选辅助型。不高声喊话，低头对人说两句，下一个 | 记录每次光屏弹出的可选能力列表 | +8=22 |
| R3 | 破甲一击+/龟壳防御+/暴击瞄准 | 选攻击型。打出升级版侦查眼（揭示弱点+双倍暴击）→击中守夜人守护兽眼睛→关键伤害 | 用控制能力封住破晓者守护兽攻击窗口→队友完成击杀 | 记录→发现陈迹攻击模式异常 | +8=30 |

**R3关键节点**：
- 陈迹伤害比预期高三倍。系统判定"关键伤害，获得积分+400"
- 同一毫秒，白羽让守夜人队友完成对破晓者守护兽最后一击
- **破晓者守护兽倒下** → 守夜人胜

**判定与结算**
- 破晓者约35人面临死亡。人群开始一盏一盏消失
- 陈迹任务积分累计约750（侦查+200、告知弱点+150、关键伤害+400），自动扣500触发死亡豁免，胸口浮银光转移至守夜人。剩余约250积分——仍是全大厅最穷的玩家之一
- 另有2人积分也够豁免门槛（一笔带过即可）
- 最终：守夜人35 + 3豁免 = **38人存活**
- 老韩也消失了——但幸存者名单上他出现在第41名。陈迹注意到异常

**道具竞拍**（3件）

| 道具 | 类型 | 效果 | 获得者 | 牌路 |
|------|------|------|--------|------|
| 数据记录 | 持续 | 回看任一已结束回合的详细数据 | **老韩** | 洞察① |
| 定身符 | 消耗3 | 定身一名玩家一回合 | **白羽** | 守护① |
| 空白牌 | 不明 | "它什么都不是。除非你相信它是某样东西。" | **陈迹** | 悖逆① |

**羁绊线索**：空白牌背面有极小圈中横线印记（悖逆之路标记）。数据记录的菱形纹路在靠近空白牌时有微弱共鸣——两人当时不认识，没注意到。

### 5.3 第2局：双塔之争

**基本信息**
- 阵营：曦月 vs 暗星（各19人）
- 胜利条件：先将己方塔升至第10层
- 机制：每回合派遣玩家——可去己方塔采集升级材料，或去对方塔进行破坏
- 塔结构：黑色玄武岩，10层，每层间有发光凹槽需材料填充

  1. 成功采集升级材料并运回己方塔 → +100~200/趟
  2. 成功破坏对方塔层 → +200/层
  3. 使用侦查能力获取对方进度/配置信息 → +200
- **获胜奖励**：胜方阵营每人 +300

**回合明细**

1. **系统的残酷是真实的。** 死的代价不可撤销。不给读者发"隐藏生路"的安慰剂
2. **希望不来自规则放水。** 来自角色在不可逆的损失后仍然选择往前走
3. **没人能单独赢。** 陈迹的策略+白羽的判断+老韩的数据——缺一不可
4. **"回家"的两种含义。** 庄闲不回家→回不去了。老韩想回家→回不去。陈迹白羽回去了，口袋里多了两张牌
5. **信任是算出来的，也是给出去的。** 第5局不靠口号——靠"前三积分+输了陪你们一起死"。让人可以算的信任，才是有效的信任
